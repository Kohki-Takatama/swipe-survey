🚀 GAS デプロイ手順

1
スプレッドシートを開く
記録先のGoogle スプレッドシートを開き、メニューから 拡張機能 → Apps Script をクリック。
拡張機能›Apps Script
アドオン
マクロ
⚡ Apps Script
AppSheet
2
コードを貼り付けて保存
エディタのコードを全選択して削除し、下のコードをコピーして貼り付け。Ctrl+S（Cmd+S） で保存。
gas-doPost.gs

        
3
ウェブアプリとしてデプロイ
デプロイ → 新しいデプロイ をクリックし、以下の設定で作成。
種類
📄 ウェブアプリ
次のユーザーとして実行
自分
アクセスできるユーザー
全員

        
4
ウェブアプリ URL をコピー
デプロイ完了後、ウェブアプリの URL をコピー。
ウェブアプリ › URL
📋 コピー ← ここ

        
5
URL を Survey Builder に貼り付け
コピーした URL を上部の 「GAS URL」 フィールドに貼り付けてHTMLをダウンロード。
クラウド版は初回起動時にアプリ内で設定できます。
